• JavaScript | Events

Mouse Events

click : Bir öğeye tıklanıldığında tetiklenir.

dblclick : Bir öğeye çift tıklandığında tetiklenir.

mousedown : Fare düğmesine basıldığında tetiklenir.

mouseup : Fare düğmesi bırakıldığında tetiklenir.

mousemove : Fare hareket ettirildiğinde tetiklenir.

mouseenter : Fare, öğenin içine girdiğinde tetiklenir (bubbling yok).

mouseleave : Fare, öğeden çıktığında tetiklenir (bubbling yok).

mouseover : Fare, öğenin üzerine geldiğinde tetiklenir.

mouseout : Fare, öğeden çıktığında tetiklenir.

contextmenu : Kullanıcı sağ tıkladığında tetiklenir.

Keyboard Events

keydown : Klavye tuşuna basıldığında tetiklenir.

keypress : (Artık kullanılmıyor) Karakter girdisi olduğunda tetiklenirdi.

keyup : Klavye tuşu bırakıldığında tetiklenir.

Form/Input Events

input : Kullanıcı input alanına veri girdiğinde tetiklenir.

reset : Form sıfırlandığında tetiklenir.

change : Input veya select gibi alanlarda değer değiştiğinde tetiklenir.

blur : Odak kaybolduğunda tetiklenir.

focusout : Bir alt öğe odaktan çıktığında tetiklenir (bubbling var).

focus : Öğe odaklandığında tetiklenir.

focusin : Bir alt öğe odaklandığında tetiklenir (bubbling var).

submit : Bir form gönderildiğinde tetiklenir.

invalid : Form doğrulaması başarısız olduğunda tetiklenir.

Clipboard Events

copy : Kullanıcı içerik kopyaladığında tetiklenir.

cut : Kullanıcı içerik kestiğinde tetiklenir.

paste : Kullanıcı içerik yapıştırdığında tetiklenir.

Drag and Drop Events

drag : Öğe sürüklenirken tekrar tekrar tetiklenir.

dragenter : Sürüklenen öğe hedefin üzerine geldiğinde tetiklenir.

dragend : Sürükleme bittiğinde tetiklenir.

dragover : Sürüklenen öğe hedefin üzerinde hareket ettikçe tetiklenir.

dragleave : Sürüklenen öğe hedefin dışına çıktığında tetiklenir.

dragstart : Sürükleme başladığında tetiklenir.

drop : Sürüklenen öğe hedefe bırakıldığında tetiklenir.

Window Events

load : Sayfa tamamen yüklendiğinde tetiklenir.

unload : Sayfa kapatılırken tetiklenir.

beforeunload : Kapanmadan hemen önce kullanıcıyı uyarır.

resize : Pencere boyutu değiştirildiğinde tetiklenir.

scroll : Kullanıcı sayfada kaydırma yaptığında tetiklenir.

error : Bir hata oluştuğunda tetiklenir.

hashchange : URL'deki hash değeri değiştiğinde tetiklenir.

popstate : Tarayıcı geçmişi değiştiğinde tetiklenir (history API ile).

storage : Aynı domain'deki başka sekmede localStorage değiştiğinde tetiklenir.

online : Tarayıcı çevrimiçi olduğunda tetiklenir.

offline : Tarayıcı çevrimdışı olduğunda tetiklenir.

Media Events

play : Medya oynatılmaya başladığında tetiklenir.

pause : Medya duraklatıldığında tetiklenir.

ended : Medya bittiğinde tetiklenir.

volumechange : Medya bittiğinde tetiklenir.

timeupdate : Oynatma zamanı güncellendiğinde tetiklenir.

seeking : Yeni konuma atlama başladığında tetiklenir.

seeked : Yeni konuma atlama tamamlandığında tetiklenir.

loadeddata : Veri yüklendiğinde tetiklenir.

canplay : Medya oynatılabilir hale geldiğinde tetiklenir.

canplaythrough : Medya takılmadan oynatılabilecek hale geldiğinde tetiklenir.

durationchange : Medyanın süresi değiştiğinde tetiklenir.

ratechange : Oynatma hızı değiştiğinde tetiklenir.

waiting : Medya duraksadığında tetiklenir (buffering).

Animation Events

animationstart : CSS animasyonu başladığında tetiklenir.

animationend : CSS animasyonu sona erdiğinde tetiklenir.

animationiteration : CSS animasyonu bir turu tamamladığında tetiklenir.

Transition Events

transitionstart : CSS geçişi başladığında tetiklenir.

transitionend : CSS geçişi bittiğinde tetiklenir.

transitionrun : Geçiş başladığında (tam devreye girmeden) tetiklenir.

transitioncancel : Geçiş iptal edildiğinde tetiklenir.

Touch Events

touchstart : Bir parmak ekranla temas ettiğinde tetiklenir.

touchmove : Parmak ekran üzerinde hareket ettirildiğinde tetiklenir.

touchend : Parmak ekranla teması kestiğinde tetiklenir.

touchcancel : Sistem, dokunmayı iptal ettiğinde tetiklenir (ör. çağrı geldiğinde).

Pointer Events

pointerover : İşaretçi öğenin üzerine geldiğinde tetiklenir.

pointerenter : İşaretçi öğeye girdiğinde tetiklenir (bubbling yok).

pointerdown : İşaretçi ile öğeye basıldığında tetiklenir.

pointermove : İşaretçi hareket ettiğinde tetiklenir.

pointerup : İşaretçi bırakıldığında tetiklenir.

pointercancel : İşaretçi iptal edildiğinde tetiklenir.

pointerout : İşaretçi öğeden çıktığında tetiklenir.

pointerleave : İşaretçi öğeyi terk ettiğinde tetiklenir (bubbling yok).

Diğer (Misc)

wheel : Fare tekerleği döndürüldüğünde tetiklenir.

deviceorientation : Cihaz yönü değiştiğinde tetiklenir.

devicemotion : Cihaz hareket ettiğinde tetiklenir.

fullscreenchange : Tam ekran modu değiştiğinde tetiklenir.

fullscreenerror : Tam ekran modu hatası oluştuğunda tetiklenir.

visibilitychange : Sayfanın görünürlük durumu değiştiğinde tetiklenir (arka plana geçtiğinde).

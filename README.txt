/* JS|Events */

Bu proje, JavaScript’teki tüm event türlerini (olayları) açıklamalarla öğrenmeyi kolaylaştırmak amacıyla hazırlanmıştır. Her bir event türü teorik olarak açıklanmıştır.

* Amaç : 
  + Event türlerini kapsamlı şekilde tanıtmak
  + Sık kullanılan event türlerini (click, input, submit, keydown, vs.) öğretmek
  + Yeni başlayanlar için sade, anlaşılır bir kaynak sunmak

* İçerik :
  + Mouse Events

  + Form/Input Events
  + Clipboard Events
  + Drag and Drop Events
  + Window Events
  + Media Events
  + Animation Events
  + Transition Events
  + Touch Events
  + Pointer Events
  + Diğer (Misc)

/* JS|Events */

This project is designed to make it easier to learn all event types in JavaScript with explanations. Each event type is explained theoretically.

* Purpose:

 + To teach frequently used event types (click, input, submit, keydown, etc.)
 + To provide a simple, understandable resource for beginners

* Content:

 + Keyboard Events
 + Form/Input Events

 + Drag and Drop Events
